- Email & Password Authentication
- Google Sign-In
- Facebook Login ✅
- Biometric (Fingerprint) Login
- Secure Token Storage
- Dark Mode Toggle (persists in settings) ✅

- Flutter
- Firebase Authentication
- Provider (State Management)
- flutter_secure_storage
- local_auth
- Google Sign-In
- Facebook Login SDK
